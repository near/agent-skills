# NEAR Intents 1Click API Skill

Cross-chain token swaps via the 1Click API.

## Quick Start

1. Read `SKILL.md` for overview and rule index
2. Read specific `rules/*.md` files for code examples
3. Or read `AGENTS.md` for everything compiled

## Structure

```
near-intents/
├── SKILL.md      # Index - start here
├── AGENTS.md     # Full compiled rules
├── rules/
│   ├── api-*.md        # API endpoints
│   ├── deposit-*.md    # Chain deposits
│   ├── react-*.md      # React patterns
│   └── *.md            # Advanced
└── README.md
```
